
COA PDF Files Directory
======================

This directory is where you should place your PDF files for processing.

Instructions:
1. Copy all your COA PDF files into this directory
2. Supported formats: .pdf only
3. File naming: Any valid filename ending with .pdf
4. Maximum file size: 10MB per file

Example files:
- COA_BGB-21447_Batch001.pdf
- COA_BGB-16673_Lot123.pdf
- Certificate_Analysis_BGB-43395.pdf

The application will automatically scan this directory when you click
"Process PDF Files" in the Word Add-in interface.

Processing details and extracted data confidence scores will be displayed
in the console where you started the backend server.
